# codex-skills
Collection of useful and semi-useful skills for Codex.

| Skill | Description |
| --- | --- |
| `bootstrap-project` | Helps turn a rough project idea into durable repository guidance. It facilitates early decisions around scope, stack, architecture, operations, and security, then writes `AGENTS.md` and a high-level project specification. |
| `prompt-architect` | Turns rough prompt ideas, draft prompts, or custom GPT concepts into production-ready prompts. It uses a default-first clarification workflow with stable numbered decisions before rendering the final prompt. |
